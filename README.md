# 糖葫芦遥控器 (Tanghulu Remote)

欢迎使用 **糖葫芦遥控器**。这是一款专为智能电视和 Apple TV 设计的增强型遥控与多屏互动应用，旨在为您提供更流畅、便捷的大屏控制和多媒体娱乐体验。

## 核心功能介绍

1. **智能遥控与交互**
   - 完美适配设备交互，提供流畅的滑动光标、精准点击与快速导航。
   - 支持物理遥控器与应用端手势操作的无缝切换。
   - **移动与点击**：通过滑动移动大屏光标，轻松点击选择目标。
   - **快捷返回**：一键返回上一页或退出当前应用场景。

2. **多屏互动与浏览控制**
   - 深度集成大屏网页浏览控制机制，让您在电视上也能获得如同 PC 般的网页冲浪体验。
   - 支持常用网站的书签记录与快速入口。

3. **历史与缓存管理**
   - 自动记录操作与浏览历史，并可在本地进行查看或清理，保护您的隐私同时方便找回记录。

## 部署与运行

1. **环境依赖**
   - Xcode 15+ 及最新版 macOS。
   - 支持 iOS/watchOS 及 tvOS 最新系统版本。

2. **如何编译**
   - 使用 Xcode 打开 `.xcworkspace`。
   - 选择对应的应用 Target。
   - 配置好您的开发者证书 (Signing & Capabilities)。
   - 选择您的目标测试设备（iPhone/Apple TV 或模拟器），点击 `Run` (Cmd + R)。
