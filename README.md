## ✨ 核心功能

### 🎮 智能遥控与交互
* **无缝切换**：支持物理遥控器与应用端手势操作的无缝切换，完全贴合设备原生交互。
* **精准光标**：通过在手机或手表上滑动来移动大屏光标，轻松实现精准点击与选择。
* **快捷导航**：提供一键返回上一页、退出当前应用场景等便捷的系统级操作指令。

### 🌐 多屏互动与浏览控制
* **大屏冲浪**：深度集成大屏网页浏览控制机制，让您在电视上也能获得如同 PC 般随心所欲的网页冲浪体验。
* **快速书签**：支持常用网站的云端/本地书签记录，快速触达您最喜爱的网页内容。

### 🛡️ 历史与缓存管理
* **隐私保护**：自动记录操作与浏览历史，并支持在本地快速查看或一键清理，全面保护您的隐私数据，同时方便找回浏览记录。

## 🛠️ 部署与运行

如果您是开发者并希望自行编译运行，请遵循以下步骤：

### 1. 环境依赖
- **IDE**: Xcode 15.0 及以上版本
- **OS**: 最新版 macOS
- **System**: 支持 iOS 15.0+ / watchOS 8.0+ / tvOS 15.0+

### 2. 本地编译
```bash
# 1. 克隆项目或在 Xcode 中打开工程文件
open .xcworkspace

# 2. 选择对应的 Target (如 iOS App 或 tvOS App)
# 3. 在 "Signing & Capabilities" 中配置好您的开发者证书
# 4. 连接您的测试设备（iPhone / Apple TV）或使用模拟器
# 5. 点击 Run 或使用快捷键 Cmd + R
```
